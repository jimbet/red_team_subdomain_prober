# red_team_subdomain_prober
Advanced Red Team Subdomain With Common Port Prober

<br><br>

How to execute:

```
git clone https://github.com/jimbet/red_team_subdomain_prober.git
cd red_team_subdomain_prober
chmod +x jim_red-team_prober.py
chmod +x requirements.sh
bash requirements.sh
python jim_red-team_prober.py
```
